# 块级作用域

## 块级作用域存在的必要性？

函数内部的变量定义会覆盖掉外部的变量定义

```javascript
var time = new Date();

function fun(){
    console.log(time);
    if(false){
        var time = "Hello World!";
    }
}
fun();  // undefined



var string = 'Hello World!';
for(var i = 0; i < string.length; i++){
    console.log(string[i]);
}
console.log("end loop, i = " + i);
// 定义的i并不会消失，下一次重新var出来的i也依旧会存在，浪费内存空间,i的地址指向改变，内存不会释放
```

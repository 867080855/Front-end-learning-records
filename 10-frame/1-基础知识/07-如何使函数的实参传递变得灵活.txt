    如何做到一个函数明明定义了两个甚至多个参数却只需要传递一个或者少数的参数而函数不报错？
    或者，如何做到一个函数明明没有定义参数，却可以传递很多参数进去？
    ================================================================================
        传递少数的参数：
            在函数内部利用完美法则做好预防，提前声明
            if(param1) {
                // 语句体
            }
        传递无限的参数：
            利用arguments解决
            var params = arguments;
            for( var k in params){
                // 语句体
            }
